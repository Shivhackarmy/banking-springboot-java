# Banking System - Spring Boot Project

## 📌 Project Description

This is a Banking System application developed using Spring Boot. It allows users to perform basic banking operations such as account creation, deposit, withdrawal, and transaction management.

## 🚀 Features

* Create Bank Account
* Deposit Money
* Withdraw Money
* Check Account Balance
* Transaction History

## 🛠️ Tech Stack

* Java
* Spring Boot
* Spring Data JPA
* MySQL
* Maven

## 📂 Project Structure

* `src/main/java` → Main application code
* `src/main/resources` → Config files
* `pom.xml` → Dependencies

## ▶️ How to Run

1. Clone the repository
2. Open in Eclipse / IntelliJ
3. Configure database in `application.properties`
4. Run the project
